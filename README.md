# RevOps Consulting Website

A React + Vite marketing site for promoting a Revenue Operations consulting service.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy to Vercel

1. Push this repo to GitHub.
2. Open [Vercel](https://vercel.com/new) and import the repository.
3. Keep the detected **Vite** settings.
4. Use `npm run build` as the build command.
5. Use `dist` as the output directory.
6. Click **Deploy**.

## Customize before launch

- Update the placeholder email links in `src/App.jsx`
- Replace the brand name and copy with your own service details
- Add your calendar link, testimonials, and case studies
